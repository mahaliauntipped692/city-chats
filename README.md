Мы проводим оффлайн-встречи (митапы) в городах от Москвы до Буэнос-Айреса.

**Coffee&Code** is an international IT community that brings together developers, designers, and tech enthusiasts worldwide.
We host offline meetups in cities from Moscow to Buenos Aires.
